# Respi Project Website

## Getting started

```shell
# optional create a git namespace dir and cd into it
mkdir -p $HOME/src/github.com/respi-project
cd $HOME/src/github.com/respi-project

# clone the repo
git clone git@github.com:respi-project/website.git
cd respi
```
